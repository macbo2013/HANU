# HANU 社交软件安装包

项目名：HANU  
类型：PHP + MySQL 5.7 社交软件安装包  
结构：根目录安装器 + source 源码文件夹 + language 语言包文件夹

## 你要上传的内容

把本文件夹里的内容上传到服务器网站根目录：

```text
index.php
source/
language/
README.md
```

然后打开你的域名或服务器 IP：

```text
http://你的域名/
```

会自动运行根目录的 `index.php` 安装器。

## 安装器会做什么

1. 显示欢迎页
2. 检测 PHP / PDO MySQL / 目录权限
3. 选择语言：简体中文 / 繁体中文 / English
4. 填写 MySQL 5.7 数据库信息
5. 设置网站信息
6. 创建最高管理员
7. 创建权限等级 / 成员组
8. 创建默认板块
9. 自动导入 MySQL 数据表
10. 自动把 `source/` 源码复制到网站根目录
11. 自动写入 `config/config.php`
12. 自动复制语言包到 `data/lang/`
13. 自动生成 `data/install.lock`

## 安装完成后删除

安装完成后删除以下安装文件：

```text
index.php
source/
language/
README.md
```

安装器最后会提示你删除。

## 注意

- 跳转全部使用相对路径。
- 不写死域名。
- 可以放在服务器根目录，也可以放在子目录。
- 管理员就是最高权限，没有超级管理员。


## v2 权限修复说明

如果安装检测中出现 `source/data 可写 失败`，这是旧版安装器检测过严导致的。

v2 已修复：

- 不再要求 `source/data` 必须可写。
- 安装时自动创建根目录：
  - `config/`
  - `data/`
  - `data/cache/`
  - `data/uploads/`
  - `data/lang/`
- 如果服务器仍提示不可写，请在服务器文件管理器里把这些目录权限改成 755、775 或 777。

## v4 mbstring 兼容修复

修复 PHP 未安装 mbstring 扩展时报错：

```text
Fatal error: Call to undefined function mb_substr()
```

v4 中已经加入 `text_cut()` 兼容函数：

- 有 mbstring：使用 `mb_substr`
- 没有 mbstring：自动退回 `substr`

已修复：
- `app/register.php`
- `api/send.php`
- 其他直接使用 `mb_substr` 的源码位置
